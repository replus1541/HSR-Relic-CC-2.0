## Policy

- Lv15 is only the hard system cap. It is not used as the default max-damage level.
- Default max levels are basic attack Lv6 and combat skill / ultimate / talent Lv10.
- E3/E5 level bonuses are parsed per character and per skill category, then applied to the default max level.
- If a primary leveled ability has no matched coefficient row, the effect is blocked instead of using the level-1 HoyoWiki description value.
