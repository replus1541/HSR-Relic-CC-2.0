# ui-reuse

기존 UI에서 재사용할 수 있는 스타일과 컴포넌트 후보를 기록합니다.

Phase 2에서 파일 단위 재사용/참고/금지 맵을 작성합니다.

## 기록 기준

- 기존 UI 스타일과 사용 흐름은 우선 참고 대상으로 분류합니다.
- CSS나 컴포넌트를 대량 복사하지 않고, 필요한 패턴만 v2 UI layer로 재구성합니다.
- UI는 `CombatLedger`와 aggregation result만 표시해야 하므로, 계산값을 조립하는 기존 UI 로직은 재사용 금지 후보로 분류합니다.
- 모바일 기준 레이아웃, 카드 밀도, 탭/패널 흐름, trace 표시 방식은 재사용 후보로 검토합니다.
